📊 Data Analysis Projects

A collection of data analysis projects built for practice, learning, and skill development.

This repository documents my journey of learning how to work with data — from raw datasets and databases to SQL analysis, Python-based data manipulation, visualization, and extracting meaningful insights.

This repository is primarily a practice and learning repository. It may not remain actively maintained forever as my focus gradually moves toward Machine Learning and MLOps.

🔄 Typical Project Workflow

A project may follow a workflow such as:

1. Understand the problem
2. Find or collect a dataset
3. Inspect the data
4. Design a database if required
5. Load data into MySQL
6. Query and analyze data using SQL
7. Clean and transform data using Pandas
8. Perform exploratory data analysis
9. Create visualizations
10. Extract useful insights
11. Document the findings
